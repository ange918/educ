## CURRENCY & DATE FORMATTING

Create `lib/utils.ts`:

```typescript
export function formatXOF(amount: number): string {
  return new Intl.NumberFormat('fr-FR').format(amount) + ' XOF'
}

export function formatDate(dateStr: string): string {
  return new Date(dateStr).toLocaleDateString('fr-FR', {
    day: 'numeric', month: 'long', year: 'numeric'
  })
}

export function getInitials(name: string): string {
  return name.split(' ').map(n => n[0]).join('').toUpperCase().slice(0, 2)
}

export function getAvatarColor(name: string): string {
  const colors = ['#2563EB', '#10B981', '#F59E0B', '#EF4444', '#8B5CF6', '#06B6D4']
  const index = name.charCodeAt(0) % colors.length
  return colors[index]
}
```

## IMPORTANT IMPLEMENTATION RULES

1. Every API call must be routed through `lib/api.ts` — never call fetch directly in a component
2. All mock data lives exclusively in `lib/mockData.ts`
3. Use `localStorage` to simulate session: on login store `JSON.stringify({ role, name, email, token })` under key `edusecure_session`, read it in layout files to get user info
4. Each layout (`parent/layout.tsx`, `admin/layout.tsx`) must read localStorage on mount and redirect to `/` if session is missing or role doesn't match
5. The app must be 100% navigable with mock data — no empty states, no broken pages on first load
6. Currency: always use `formatXOF()` from `lib/utils.ts`
7. Dates: always use `formatDate()` from `lib/utils.ts` or French locale formatting
8. All form inputs must have proper `<label>` elements and French placeholder text
9. No external UI component library — only Tailwind CSS + Boxicons + custom components
10. TypeScript strict mode — all component props must be fully typed with interfaces
11. Use `'use client'` directive only in components that use hooks, event handlers, or browser APIs
12. All server components (pages that only display data) should remain server components
13. The student table on `/admin/students` must render all students grouped by class when "Toutes" tab is active, and filter to one class when a specific tab is clicked — this filtering must be done in client-side state, not by re-fetching
14. Format all numbers with thousand separators (45 750 000, not 45750000)
15. The welcome page (`/`) must never show the sidebar — it is a standalone full-screen page
