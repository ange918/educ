# ÉduSecure

Système de gestion scolaire avec contrôle d'accès QR pour établissements francophones.

## Run & Operate

- `npm run dev` — démarrer le serveur de développement (port 5000)
- `npm run build` — construire pour la production
- `npm run start` — démarrer en mode production (port 5000)

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS 3
- Boxicons (via CDN)
- Google Fonts : Syne (display) + DM Sans (body)

## Where things live

- `app/` — routes Next.js (App Router)
- `components/` — composants React réutilisables
- `lib/` — utilitaires, mock data, API
